## How to use - local
### Clone from github
```
git clone https://github.com/kkan0615/image-downloader-chrome-extention.git
```
### Build the code
```bash
npm run build # npm
yarn run build # yarn
```
### Move to following link
<a href="chrome://extensions/"> move to extensions </a>

### Able the developer mode

### Press the Load unpacked.
